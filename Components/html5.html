HTML5

HTML розшифровується як Hyper Text Markup Language. Він використовується для розробки веб-сторінок за допомогою мови розмітки. HTML — це абревіатура від Hypertext and Markup language. Гіпертекст визначає зв'язок між веб-сторінками. Мова розмітки використовується для визначення текстового документа в тегу, який визначає структуру веб-сторінок. HTML5 є п'ятою і поточною версією HTML. Він покращив розмітку, доступну для документів, і представив інтерфейс програмування прикладних програм (API) і об’єктну модель документа (DOM).
